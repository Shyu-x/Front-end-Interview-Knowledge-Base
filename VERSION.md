# PrepareFor 版本管理

> 本项目使用语义化版本号 (Semantic Versioning)
> 作者: Shyu

## 版本格式

```
主版本号.次版本号.修订号
MAJOR.MINOR.PATCH
```

- **MAJOR (主版本号)**: 包含不兼容的API变更
- **MINOR (次版本号)**: 向后兼容的新功能
- **PATCH (补丁版本号)**: 向后兼容的问题修复

## 当前版本

```
1.3.0
```
