## 一、哈希表

### 1.1 两数之和 (#1)
**核心思路**: 使用 HashMap 存储遍历过的元素及其下标
- 遍历数组，对于每个元素 `num`
- 检查 `target - num` 是否在 HashMap 中
- 存在则返回下标，不存在则将当前元素存入 HashMap
- **时间复杂度**: O(n)，**空间复杂度**: O(n)

### 1.2 字母异位词分组 (#49)
**核心思路**: 将字符串排序作为 HashMap 的 key
- 对每个字符串排序，排序后的字符串作为 key
- 相同 key 的字符串归为一组
- **时间复杂度**: O(n * klogk)，k 为字符串平均长度

### 1.3 最长连续序列 (#128)
**核心思路**: 使用 HashSet 去重 + 遍历
- 将所有数字存入 HashSet
- 遍历每个数字，如果它是序列起点（num-1 不在 set 中）
- 从起点开始向后遍历，记录序列长度
- **时间复杂度**: O(n)，**空间复杂度**: O(n)

### 1.4 赎金信 (#383)
**核心思路**: 字符计数数组
- 统计 magazine 中每个字符出现的次数
- 遍历 ransomNote，相应减去计数
- 如果有字符计数小于 0，返回 false

### 1.5 同构字符串 (#205)
**核心思路**: 双向映射
- 使用两个 HashMap 建立 s→t 和 t→s 的映射
- 遍历字符串，检查每个字符的映射是否一致

### 1.6 有效字母异位词 (#242)
**核心思路**: 字符计数
- 统计 s 中每个字符出现次数
- 遍历 t，相应减去计数
- 所有计数为 0 则返回 true

### 1.7 快乐数 (#202)
**核心思路**: 快慢指针检测循环 + 哈希表检测重复
- 反复计算各位平方和
- 如果结果为 1 返回 true
- 如果出现重复结果返回 false

### 1.8 存在重复元素 II (#219)
**核心思路**: 滑动窗口 + HashMap
- 维护大小为 k 的窗口
- 使用 HashMap 记录元素最新下标
- 如果元素在窗口内已存在，返回 true

## 二、双指针

### 2.1 移动零 (#283)
**核心思路**: 双指针法
- 指针 `left` 指向已处理区域的末尾（最后一个非零元素）
- 指针 `right` 遍历数组
- 遇到非零元素与 left 交换，left++
- **时间复杂度**: O(n)，**空间复杂度**: O(1)

### 2.2 盛最多水的容器 (#11)
**核心思路**: 对撞指针
- 左右指针指向两端
- 移动较短边的指针（因为盛水量由较短边决定）
- 更新最大面积
- **时间复杂度**: O(n)

### 2.3 三数之和 (#15)
**核心思路**: 排序 + 双指针
- 先对数组排序
- 固定一个指针，双指针找另外两个数
- 注意去重：跳过相同的答案
- **时间复杂度**: O(n²)

### 2.4 验证回文串 (#125)
**核心思路**: 双指针夹逼
- 左右指针从两端向中间移动
- 跳过非字母数字字符
- 比较字符（转小写）
- 不相等返回 false

### 2.5 判断子序列 (#392)
**核心思路**: 双指针
- 两个指针分别指向 s 和 t
- 匹配成功则同时移动 s 和 t 的指针
- 只移动 t 的指针
- s 遍历完返回 true

### 2.6 两数之和 II (#167)
**核心思路**: 对撞指针
- 左右指针指向首尾
- 根据和与 target 的大小关系移动指针
- **时间复杂度**: O(n)

## 三、滑动窗口

### 3.1 无重复字符的最长子串 (#3)
**核心思路**: 滑动窗口 + 字符位置记录
- 使用 HashMap 记录每个字符最后出现的位置+1
- 当遇到重复字符时，移动 left 到重复字符位置的下一位
- 更新最大长度
- **时间复杂度**: O(n)

### 3.2 找到字符串中所有字母异位词 (#438)
**核心思路**: 固定大小滑动窗口 + 字符计数
- 维护大小为 p.length 的窗口
- 使用两个计数数组比较
- 窗口滑动时更新计数

### 3.3 最小覆盖子串 (#76)
**核心思路**: 滑动窗口 + 字符计数
- 扩大 right 指针直到包含所有目标字符
- 收缩 left 指针找到最小窗口
- 记录最优结果，继续滑动

### 3.4 长度最小的子数组 (#209)
**核心思路**: 滑动窗口 + 前缀和/双指针
- 右指针扩展窗口，计算和
- 当和 >= target 时，收缩左指针
- 记录最小长度

### 3.5 和为 K 的子数组 (#560)
**核心思路**: 前缀和 + 哈希表
- 遍历数组，计算前缀和
- 查找 prefixSum - k 在哈希表中出现的次数
- 将当前前缀和加入哈希表

### 3.6 滑动窗口最大值 (#239)
**核心思路**: 单调递减队列
- 维护一个存储索引的单调递减队列
- 队首元素是当前窗口最大值
- 队尾元素比新元素小则出队
- **时间复杂度**: O(n)

## 四、普通数组

### 4.1 最大子数组和 (#53)
**核心思路**: 动态规划 / Kadane 算法
- `dp[i]` = 以第 i 个元素结尾的最大子数组和
- `dp[i] = max(nums[i], dp[i-1] + nums[i])`
- 记录全局最大值
- **空间优化**: O(1)

### 4.2 合并区间 (#56)
**核心思路**: 排序 + 遍历
- 按区间起始点排序
- 遍历区间，与前一个合并或新建区间
- **时间复杂度**: O(nlogn)

### 4.3 轮转数组 (#189)
**核心思路**:
- 方法1: 翻转三次（局部+整体+局部）
- 方法2: 环形替换（数学方法）
- 方法3: 额外数组

### 4.4 除了自身以外数组的乘积 (#238)
**核心思路**: 左右乘积数组
- 构建左侧所有元素的乘积数组
- 构建右侧所有元素的乘积数组
- 相乘得到结果
- **空间优化**: O(1)

### 4.5 缺失的第一个正数 (#41)
**核心思路**: 原地哈希
- 将数组本身作为哈希表
- 将 1~n 范围的数放到对应位置
- 遍历找到第一个位置不对应的数
- **时间复杂度**: O(n)，**空间复杂度**: O(1)

### 4.6 环形子数组的最大和 (#918)
**核心思路**: 求最大子数组和 + 分类讨论
- 情况1：普通最大子数组（不用首尾）
- 情况2：首尾相连（总和 - 最小子数组）
- 取两者最大

## 五、矩阵

- 方法1: 用第一行和第一列作为标记

### 5.2 螺旋矩阵 (#54)

- 按右、下、左、上的顺序遍历
- 每条边遍历完后收缩边界
- 当边界相遇时停止

### 5.3 旋转图像 (#48)
**核心思路**:
- 方法1: 原地旋转 = 转置 + 翻转
- 方法2: 四个位置环形替换

### 5.4 搜索二维矩阵 II (#240)
**核心思路**: 从右上/左下开始
- 从右上角开始，左下角结束
- 根据大小关系移动指针
- **时间复杂度**: O(m+n)

### 5.5 有效数独 (#36)
**核心思路**: 行列和九宫格标记

## 六、链表

### 6.1 相交链表 (#160)
**核心思路**: 双指针遍历
- 方法1: A走完走B，B走完走A，相遇点即为交点
- 方法2: 先求长度差，长链表先走差值再同时走

### 6.2 反转链表 (#206)
**核心思路**: 迭代 / 递归
- 迭代：保存 next，reverse 指向前一个
- 递归：反转后续，修改指针

### 6.3 回文链表 (#234)
**核心思路**: 快慢指针找中点 + 反转后半部分
- 快慢指针找到中点
- 反转后半部分
- 比较前后两部分

### 6.4 环形链表 (#141)
**核心思路**: 快慢指针
- 快指针一次走两步，慢指针一次走一步
- 有环必相遇

### 6.5 环形链表 II (#142)
**核心思路**: 快慢指针 + 数学推导
- 第一次相遇后，从头节点和相遇点同时走
- 再次相遇点即为入环点
- 数学：a = c + (n-1)(b+c)

### 6.6 合并两个有序链表 (#21)
**核心思路**: 迭代 / 递归
- 比较两个链表头节点，小的加入结果
- 递归：返回较小节点，递归连接

### 6.7 两数相加 (#2)
**核心思路**: 链表模拟加法
- 遍历两个链表，模拟加法
- 处理进位
- 处理最后可能的进位

### 6.8 删除倒数第N个节点 (#19)
**核心思路**: 双指针 + 哑节点
- 快指针先走 n 步
- 然后快慢指针同时走
- 快指针到末尾时，慢指针指向删除节点前一个

### 6.9 两两交换链表中的节点 (#24)
**核心思路**: 迭代模拟
- 保存要交换的两个节点和前驱
- 修改指针关系
- 更新指针位置

### 6.10 K个一组翻转链表 (#25)
**核心思路**: 计数 + 翻转
- 每 k 个节点一组
- 翻转每组
- 连接翻转后的各组
- **关键**: 记录每组的前驱和后继

### 6.11 随机链表的复制 (#138)
**核心思路**: 哈希表 / 原地复制
- 方法1: 两次遍历 + 哈希表
- 方法2: 原地复制（复制节点插入原节点后）

### 6.12 排序链表 (#148)
**核心思路**: 归并排序
- 找中点（快慢指针）
- 递归排序两个子链表
- 合并两个有序链表

### 6.13 合并K个升序链表 (#23)
**核心思路**: 堆 / 分治
- 方法1: 最小堆（优先队列）
- 方法2: 两两合并（分治）

### 6.14 LRU缓存 (#146)
**核心思路**: 哈希表 + 双向链表
- 哈希表：O(1) 查找
- 双向链表：O(1) 移动到头部

## 七、二叉树

### 7.1 二叉树的中序遍历 (#94)
**核心思路**: 递归 / 迭代 + 栈
- 递归：左子树 → 根 → 右子树
- 迭代：用栈模拟递归

### 7.2 二叉树的最大深度 (#104)
**核心思路**: 递归 / BFS
- 递归：1 + max(左子树深度, 右子树深度)
- BFS：层序遍历计数

### 7.3 翻转二叉树 (#226)
**核心思路**: 递归 / BFS
- 递归：交换左右子树，递归处理子树
- BFS：层序遍历交换

### 7.4 对称二叉树 (#101)
**核心思路**: 递归 / 迭代
- 递归：比较左子树和右子树是否对称
- 迭代：队列或栈成对比较

### 7.5 二叉树的直径 (#543)
**核心思路**: 后序遍历
- 求每个节点的左右子树深度和
- 更新最大直径
- 返回当前节点深度

### 7.6 二叉树的层序遍历 (#102)
**核心思路**: BFS / 递归
- BFS：队列 + 按层处理
- 递归：传递层级参数

### 7.7 有序数组转BST (#108)
**核心思路**: 二分 + 递归
- 取中间元素作为根
- 递归构建左右子树

### 7.8 验证二叉搜索树 (#98)
**核心思路**: 中序遍历 + 上界下界
- BST 中序遍历是升序
- 记录上一个节点值比较
- 或传递上下界递归验证

### 7.9 BST第K小 (#230)
**核心思路**: 中序遍历 / 剪枝
- 中序遍历找到第 k 个
- 或利用 BST 性质剪枝

### 7.10 二叉树右视图 (#199)
**核心思路**: BFS / DFS
- BFS：取每层最后一个
- DFS：右子树优先，记录深度

### 7.11 二叉树展开为链表 (#114)
**核心思路**: 后序遍历 / 迭代
- 方法1: 原地展开（后续遍历）
- 方法2: 先序遍历 + 展开

### 7.12 构造二叉树 (#105)
**核心思路**: 递归 + 哈希表
- 前序找根，中序定位左右子树
- 哈希表加速中序定位

### 7.13 路径总和 III (#437)
**核心思路**: 前缀和 + 哈希表
- 记录从根到当前的前缀和
- 查找 prefixSum - target 的数量

### 7.14 二叉树最近公共祖先 (#236)
**核心思路**: 递归
- 如果找到 p 或 q，返回
- 递归左右子树
- 如果左右都找到，当前是 LCA
- 否则返回非空的那个

### 7.15 二叉树最大路径和 (#124)
**核心思路**: 后序遍历
- 计算包含当前节点的贡献值
- 更新全局最大路径和

## 八、图论

### 8.1 岛屿数量 (#200)
**核心思路**: DFS / BFS
- 遍历 grid，遇到 '1' 开始 DFS/BFS

- **时间复杂度**: O(m*n)

**核心思路**: BFS 多源
- 用队列存储腐烂橘子及其时间
- 同时向四周扩散
- 最后检查是否有未腐烂的橘子

### 8.3 课程表 (#207)
**核心思路**: 拓扑排序 / DFS 检测环
- 方法1: BFS（计算入度）
- 方法2: DFS 检测环

### 8.4 课程表 II (#210)
**核心思路**: 拓扑排序
- BFS 求拓扑序
- 检查是否有环

### 8.5 被围绕的区域 (#130)
**核心思路**: 边界DFS
- 从边界上的 'O' 开始 DFS
- 标记所有与边界相连的 'O'
- 遍历全图，将未标记的 'O' 变成 'X'

### 8.6 克隆图 (#133)
**核心思路**: BFS/DFS + 哈希表
- 哈希表存新旧节点映射
- 遍历原图，复制节点和边

### 8.7 除法求值 (#399)
**核心思路**: 并查集 / 图 + BFS
- 将等式转化为有向图
- 使用并查集或 BFS 求值

### 8.8 蛇梯棋 (#909)
**核心思路**: BFS
- 队列存储当前位置和掷骰次数
- 遍历 1-6 步
- 记录访问过的格子

### 8.9 最小基因变化 (#433)
**核心思路**: BFS
- 每次变化一个字符
- 检查是否在 bank 中

### 8.10 单词接龙 (#127)
**核心思路**: BFS / 双向BFS
- 建图：单词只差一个字符
- BFS 求最短路径

## 十三、贪心算法

### 13.1 买卖股票最佳时机 (#121)
**核心思路**: 记录最低价和最大利润
- 遍历记录最低价格
- 计算当前价格的最大利润

### 13.2 买卖股票 II (#122)
**核心思路**: 累加所有正收益
- 只要明天比今天高，今天买明天卖

### 13.3 跳跃游戏 (#55)
**核心思路**: 维护能到达的最远位置
- 遍历更新最远位置
- 能到达最后返回 true

### 13.4 跳跃游戏 II (#45)
**核心思路**: 每次在可跳跃范围内选择能跳最远的
- 维护当前边界和最远边界

### 13.5 划分字母区间 (#763)
**核心思路**: 记录每个字符最后出现位置
- 遍历确定每个片段的最远位置
- 到最远位置时分割

### 13.6 分发糖果 (#135)
**核心思路**: 两次贪心
- 从左到右：比左边高就多一个
- 从右到左：比右边高就多一个

## 十四、动态规划

### 14.1 爬楼梯 (#70)
**核心思路**: 斐波那契
- dp[i] = dp[i-1] + dp[i-2]
- 空间优化到 O(1)

### 14.2 杨辉三角 (#118)
**核心思路**: 动态规划
- 每行首尾为1
- 其他是上一行相邻之和

### 14.3 打家劫舍 (#198)
**核心思路**: DP
- dp[i] = max(dp[i-1], dp[i-2] + nums[i])
- 空间优化到 O(1)

### 14.4 完全平方数 (#279)
**核心思路**: 完全背包
- dp[i] = min(dp[i], dp[i - j*j] + 1)

### 14.5 零钱兑换 (#322)
**核心思路**: 完全背包
- dp[amount] = min(dp[amount], dp[amount - coin] + 1)

### 14.6 单词拆分 (#139)
**核心思路**: DP
- dp[i] = dp[j] && wordSet.contains(s.substring(j, i))

### 14.7 最长递增子序列 (#300)
**核心思路**: DP / 二分
- DP: O(n²)
- 二分: O(nlogn) - 维护递增序列

### 14.8 乘积最大子数组 (#152)
**核心思路**: DP
- 同时记录最大和最小（因为有负数）
- dp[i] = max/min(dp[i-1]*nums[i], nums[i])

### 14.9 分割等和子集 (#416)
**核心思路**: 01背包
- 背包容量 = sum/2
- dp[j] = dp[j] || dp[j - nums[i]]

### 14.10 最长有效括号 (#32)
**核心思路**: DP / 栈
- DP: dp[i] = dp[i-2] + dp[i-1] + 2
- 栈: 记录最后未匹配的右括号位置

### 14.11 不同路径 (#62)
**核心思路**: DP
- dp[i][j] = dp[i-1][j] + dp[i][j-1]
- 空间优化到 O(n)

### 14.12 最小路径和 (#64)
**核心思路**: DP
- dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]
- 原地修改

### 14.13 最长回文子串 (#5)
**核心思路**: 中心扩展 / DP
- 中心扩展: O(n²)
- DP: dp[i][j] = s[i]==s[j] && (j-i<3 || dp[i+1][j-1])

### 14.14 最长公共子序列 (#1143)
**核心思路**: DP
- dp[i][j] = dp[i-1][j-1] + 1 或 max(dp[i-1][j], dp[i][j-1])

### 14.15 编辑距离 (#72)
**核心思路**: DP
- 插入、删除、替换三种操作
- dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1

## 十五、位运算

### 15.1 只出现一次的数字 (#136)
**核心思路**: 异或
- 相同为0，不同为1
- 所有数字异或，结果是只出现一次的数

### 15.2 多数元素 (#169)
**核心思路**: Boyer-Moore 投票
- 候选人票数抵消
- 最后剩下的就是多数元素

### 15.3 颠倒二进制位 (#190)
**核心思路**: 位操作
- 逐位提取并组装
- n & 1 获取最低位

### 15.4 位1的个数 (#191)
**核心思路**: n & (n-1)
- 每次消除最低位的1
- 计数直到 n 为 0

### 15.5 只出现一次 II (#137)
**核心思路**: 位运算统计
- 对每一位统计1出现的次数
- 对3取模得到只出现一次的位

## 十六、数学

### 16.1 回文数 (#9)
**核心思路**: 翻转一半
- 翻转后半部分数字
- 与前半部分比较

### 16.2 加一 (#66)
**核心思路**: 从后向前处理进位
- 遇到9则变0继续
- 否则加1返回

### 16.3 平方根 (#69)
**核心思路**: 二分 / 牛顿法
- 二分查找
- 牛顿迭代法

### 16.4 Pow(x,n) (#50)
**核心思路**: 快速幂
- n 是偶数：x² 的 n/2 次方
- n 是奇数：x² 的 n/2 次方 * x
- 负指数转为 1/x

## 十七、Trie

### 17.1 实现 Trie (#208)
**核心思路**: 字典树节点
- 每个节点有 children 和 isEnd 标志
- insert: 创建/遍历节点
- search: 遍历检查 isEnd
- startsWith: 遍历检查前缀

### 17.2 单词搜索 II (#212)
**核心思路**: Trie + DFS
- 将所有单词插入 Trie
- 从每个格子开始 DFS
- 找到则加入结果
