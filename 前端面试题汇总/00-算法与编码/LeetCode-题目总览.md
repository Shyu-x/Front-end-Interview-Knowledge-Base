## 一、哈希表

### 1.1 两数之和 (Two Sum)
- **题号**: #1
- **难度**: 简单
- **题目要求**:
  - 给定一个整数数组 `nums` 和一个整数目标值 `target`
  - 请你在该数组中找出和为目标值 `target` 的那两个整数
  - 返回它们的数组下标
- **输入规范**:
  - `nums`: 整数数组，范围 `[-10^9, 10^9]`
  - `target`: 整数，范围 `[-10^9, 10^9]`
  - 每种输入只会对应一个答案，且不能使用两次相同的元素
- **输出规范**: 返回数组下标 `[i, j]`，顺序任意

### 1.2 字母异位词分组 (Group Anagrams)
- **题号**: #49
- **难度**: 中等
- **题目要求**:
  - 给定一个字符串数组，请你将字母异位词组合在一起
  - 字母异位词是由重新排列源单词的所有字母得到的新单词
- **输入规范**:
  - `strs`: 字符串数组，每个字符串由小写英文字母组成
  - 字符串长度范围 `[1, 10^4]`
- **输出规范**: 返回分组后的二维数组，每个子数组内的字符串互为字母异位词

### 1.3 最长连续序列 (Longest Consecutive Sequence)
- **题号**: #128
- **难度**: 中等
- **题目要求**:
  - 给定一个未排序的整数数组 `nums`
  - 找出数字连续的最长序列（不要求序列元素在原数组中连续）
  - 返回最长序列的长度
- **输入规范**:
  - `nums`: 整数数组，范围 `[-10^9, 10^9]`
  - 数组长度范围 `[0, 10^5]`
- **输出规范**: 返回最长连续序列的长度（整数）

### 1.4 赎金信 (Ransom Note)
- **题号**: #383
- **难度**: 简单
- **题目要求**:
  - 给定两个字符串 `ransomNote` 和 `magazine`
  - 判断 `ransomNote` 能不能由 `magazine` 里面的字符构成
  - 如果可以返回 `true`，否则返回 `false`
- **输入规范**:
  - `ransomNote`: 字符串，小写英文字母
  - `magazine`: 字符串，小写英文字母
  - 字符串长度范围 `[1, 10^5]`
- **输出规范**: 返回布尔值

### 1.5 同构字符串 (Isomorphic Strings)
- **题号**: #205
- **难度**: 简单
- **题目要求**:
  - 给定两个字符串 `s` 和 `t`
  - 判断它们是否是同构的
  - 同构意味着 `s` 中的每个字符可以被替换得到 `t`
- **输入规范**:
  - `s`, `t`: 字符串，长度相同 `[1, 10^4]`
- **输出规范**: 返回布尔值

### 1.6 有效的字母异位词 (Valid Anagram)
- **题号**: #242
- **难度**: 简单
- **题目要求**:
  - 给定两个字符串 `s` 和 `t`
  - 判断 `t` 是否是 `s` 的字母异位词
- **输入规范**:
  - `s`, `t`: 字符串，小写英文字母，长度相同 `[1, 10^5]`
- **输出规范**: 返回布尔值

### 1.7 快乐数 (Happy Number)
- **题号**: #202
- **难度**: 简单
- **题目要求**:
  - 给定一个整数 `n`
  - 编写一个算法判断该数是否是快乐数
  - 快乐数定义：反复将各位数字平方求和，最终得到 1
- **输入规范**:
  - `n`: 正整数，范围 `[1, 2^31 - 1]`
- **输出规范**: 返回布尔值

### 1.8 存在重复元素 II (Contains Duplicate II)
- **题号**: #219
- **难度**: 简单
- **题目要求**:
  - 给定一个整数数组 `nums` 和一个整数 `k`
  - 判断是否存在两个不同的索引 `i` 和 `j`
  - 使得 `nums[i] == nums[j]` 且 `|i - j| <= k`
- **输入规范**:
  - `nums`: 整数数组，长度 `[1, 10^4]`
  - `k`: 正整数，范围 `[1, 10^4]`
- **输出规范**: 返回布尔值

## 二、双指针

### 2.1 移动零 (Move Zeroes)
- **题号**: #283
- **难度**: 简单
- **题目要求**:
  - 给定一个数组 `nums`
  - 将所有 `0` 移动到数组末尾，同时保持非零元素的相对顺序
  - 不能复制数组，只能原地对数组进行操作
- **输入规范**:
  - `nums`: 整数数组，长度 `[1, 10^4]`
- **输出规范**: 无返回值，原地修改数组

### 2.2 盛最多水的容器 (Container With Most Water)
- **题号**: #11
- **难度**: 中等
- **题目要求**:
  - 给定一个长度为 `n` 的整数数组 `height`
  - 有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])`
  - 找出其中的两条线，使得它们与 x 轴共同构成的容器能容纳最多的水
- **输入规范**:
  - `height`: 非负整数数组，长度 `[2, 10^5]`
- **输出规范**: 返回最大盛水量（整数）

### 2.3 三数之和 (3Sum)
- **题号**: #15
- **难度**: 中等
- **题目要求**:
  - 给定一个整数数组 `nums`
  - 返回所有满足 `nums[a] + nums[b] + nums[c] = 0` 的三元组 `[a, b, c]`
  - 不能包含重复的三元组
- **输入规范**:
  - `nums`: 整数数组，长度 `[3, 3000]`
- **输出规范**: 返回二维数组，每个子数组为满足条件的三元组

### 2.4 验证回文串 (Valid Palindrome)
- **题号**: #125
- **难度**: 简单
- **题目要求**:
  - 给定一个字符串 `s`
  - 判断它是否是回文串
  - 考虑只 alphanumeric 字符，并忽略大小写
- **输入规范**:
  - `s`: 字符串，长度 `[0, 2 * 10^5]`
- **输出规范**: 返回布尔值

### 2.5 判断子序列 (Is Subsequence)
- **题号**: #392
- **难度**: 简单
- **题目要求**:
  - 给定字符串 `s` 和 `t`
  - 判断 `s` 是否是 `t` 的子序列
- **输入规范**:
  - `s`: 字符串，长度 `[0, 10^5]`
  - `t`: 字符串，长度 `[0, 10^4]`
- **输出规范**: 返回布尔值

### 2.6 两数之和 II - 输入有序数组 (Two Sum II)
- **题号**: #167
- **难度**: 中等
- **题目要求**:
  - 给定一个已按照升序排列的整数数组 `numbers`
  - 找出两个数使得它们相加之和等于目标数
  - 返回这两个数的下标（下标从 1 开始）
- **输入规范**:
  - `numbers`: 升序整数数组，长度 `[2, 3 * 10^4]`
  - `target`: 整数
- **输出规范**: 返回数组下标 `[index1, index2]`，1-indexed

## 三、滑动窗口

### 3.1 无重复字符的最长子串 (Longest Substring Without Repeating Characters)
- **题号**: #3
- **难度**: 中等
- **题目要求**:
  - 给定一个字符串 `s`
  - 找出其中不含有重复字符的最长子串的长度
- **输入规范**:
  - `s`: 字符串，可包含任意字符，长度 `[0, 5 * 10^4]`
- **输出规范**: 返回最长子串长度（整数）

### 3.2 找到字符串中所有字母异位词 (Find All Anagrams in a String)
- **题号**: #438
- **难度**: 中等
- **题目要求**:
  - 给定两个字符串 `s` 和 `p`
  - 找出 `s` 中所有 `p` 的字母异位词的子串
  - 返回这些子串的起始位置
- **输入规范**:
  - `s`: 字符串，小写英文字母，长度 `[1, 10^5]`
  - `p`: 字符串，小写英文字母，长度 `[1, 10^4]`
- **输出规范**: 返回起始位置数组

### 3.3 最小覆盖子串 (Minimum Window Substring)
- **题号**: #76
- **难度**: 困难
- **题目要求**:
  - 给你两个字符串 `s` 和 `t`
  - 请在 `s` 中找出涵盖 `t` 中所有字符的最小子串
- **输入规范**:
  - `s`: 字符串，可包含任意字符，长度 `[1, 10^5]`
  - `t`: 字符串，可包含任意字符，长度 `[1, 10^4]`
- **输出规范**: 返回最小覆盖子串，若不存在返回空字符串

### 3.4 长度最小的子数组 (Minimum Size Subarray Sum)
- **题号**: #209
- **难度**: 中等
- **题目要求**:
  - 给定一个含有 `n` 个正整数的数组 `nums` 和一个正整数 `target`
  - 找出连续子数组满足其和 >= target 的最小长度
  - 若不存在满足条件的子数组，返回 0
- **输入规范**:
  - `nums`: 正整数数组，长度 `[1, 10^5]`
  - `target`: 正整数
- **输出规范**: 返回最小长度（整数）

### 3.5 和为 K 的子数组 (Subarray Sum Equals K)
- **题号**: #560
- **难度**: 中等
- **题目要求**:
  - 给定一个整数数组 `nums` 和整数 `k`
  - 返回连续子数组的和等于 `k` 的数量
- **输入规范**:
  - `nums`: 整数数组，长度 `[1, 2 * 10^4]`
  - `k`: 整数，范围任意
- **输出规范**: 返回满足条件的子数组数量

### 3.6 滑动窗口最大值 (Sliding Window Maximum)
- **题号**: #239
- **难度**: 困难
- **题目要求**:
  - 给你一个整数数组 `nums`
  - 有一个大小为 `k` 的滑动窗口
  - 找出每个滑动窗口中的最大值
- **输入规范**:
  - `nums`: 整数数组，长度 `[1, 3 * 10^4]`
  - `k`: 正整数，范围 `[1, nums.length]`
- **输出规范**: 返回数组，包含每个窗口的最大值

## 四、普通数组

### 4.1 最大子数组和 (Maximum Subarray)
- **题号**: #53
- **难度**: 中等
- **题目要求**:
  - 给定一个整数数组 `nums`
  - 找出其中和最大的连续子数组（子数组最少包含一个元素）
  - 返回其最大和
- **输入规范**:
  - `nums`: 整数数组，长度 `[1, 10^5]`
  - 元素范围 `[-10^4, 10^4]`
- **输出规范**: 返回最大和（整数）

### 4.2 合并区间 (Merge Intervals)
- **题号**: #56
- **难度**: 中等
- **题目要求**:
  - 给定一个区间数组 `intervals`
  - 合并所有重叠的区间
  - 返回不重叠的区间数组
- **输入规范**:
  - `intervals`: 二维数组，每个区间 `[start_i, end_i]`
  - 区间数 `[1, 10^4]`
- **输出规范**: 返回合并后的区间数组

### 4.3 轮转数组 (Rotate Array)
- **题号**: #189
- **难度**: 中等
- **题目要求**:
  - 给定一个数组 `nums`
  - 将数组中的元素向右轮转 `k` 个位置
- **输入规范**:
  - `nums`: 整数数组，长度 `[1, 10^5]`
  - `k`: 非负整数
- **输出规范**: 无返回值，原地修改数组

### 4.4 除了自身以外数组的乘积 (Product of Array Except Self)
- **题号**: #238
- **难度**: 中等
- **题目要求**:
  - 给定一个数组 `nums`
  - 返回数组 `answer`，其中 `answer[i]` 等于 `nums` 中除 `nums[i]` 之外其余各元素的乘积
  - 不能使用除法，且 O(n) 时间复杂度
- **输入规范**:
  - `nums`: 整数数组，长度 `[2, 10^5]`
- **输出规范**: 返回数组 `answer`

### 4.5 缺失的第一个正数 (First Missing Positive)
- **题号**: #41
- **难度**: 困难
- **题目要求**:
  - 给定一个未排序的整数数组 `nums`
  - 找出其中没有出现的最小的正整数
  - 实现 O(n) 时间复杂度和 O(1) 空间复杂度
- **输入规范**:
  - `nums`: 整数数组，长度 `[1, 10^5]`
- **输出规范**: 返回缺失的第一个正整数

### 4.6 环形子数组的最大和 (Maximum Sum Circular Subarray)
- **题号**: #918
- **难度**: 中等
- **题目要求**:
  - 给定一个长度为 `n` 的环形数组 `nums`
  - 找出非空子数组的最大和
  - 环形数组首尾相连
- **输入规范**:
  - `nums`: 整数数组，长度 `[1, 2 * 10^4]`
- **输出规范**: 返回最大和

### 5.4 搜索二维矩阵 II (Search a 2D Matrix II)

- **输出规范**: 返回布尔值

### 5.5 有效的数独 (Valid Sudoku)
