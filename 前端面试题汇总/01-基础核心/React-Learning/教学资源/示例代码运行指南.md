# React 学习示例代码运行指南

> 本指南将帮助你设置开发环境并运行教学示例代码

---

## 1. 环境准备

### 1.1 安装 Node.js

```bash
# 检查是否已安装
node -v

# 如果未安装，下载安装 https://nodejs.org/
# 推荐安装 Node.js 25.x LTS 版本
```

### 1.2 安装 pnpm（推荐）

```bash
# 使用 npm 安装
npm install -g pnpm

# 验证安装
pnpm -v
```

---

## 2. 创建 React 项目

### 2.1 使用 Vite 创建项目

```bash
# 创建新的 React 项目
pnpm create vite@latest react-learning-demo -- --template react

# 进入项目目录
cd react-learning-demo

# 安装依赖
pnpm install
```

### 2.2 安装项目依赖

```bash
# 安装 React Router
pnpm add react-router

# 安装 Ant Design（如果需要）
pnpm add antd @ant-design/icons

# 安装其他依赖
pnpm add
```

---

## 3. 运行示例代码

### 3.1 启动开发服务器

```bash
# 启动开发服务器（默认端口 5173）
pnpm dev

# 指定端口
pnpm dev -- --port 3000

# 局域网访问
pnpm dev -- --host
```

### 3.2 访问应用

打开浏览器访问 http://localhost:5173

---

## 4. 添加示例代码

### 4.1 创建基础组件

在 `src/components` 目录下创建组件文件：

```jsx
// src/components/HelloWorld.jsx
function HelloWorld() {
  return (
    <div>
      <h1>Hello, World!</h1>
      <p>欢迎学习 React</p>
    </div>
  )
}

export default HelloWorld
```

### 4.2 在 App.jsx 中使用

```jsx
// src/App.jsx
import HelloWorld from './components/HelloWorld'

function App() {
  return (
    <div>
      <HelloWorld />
    </div>
  )
}

export default App
```

---

## 5. 路由配置示例

### 5.1 安装 React Router

```bash
pnpm add react-router
```

### 5.2 配置路由

```jsx
// src/main.jsx
import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import { BrowserRouter, Routes, Route } from 'react-router'
import App from './App.jsx'
import Home from './pages/Home.jsx'
import About from './pages/About.jsx'
import './index.css'

createRoot(document.getElementById('root')).render(
  <StrictMode>
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<App />}>
          <Route index element={<Home />} />
          <Route path="about" element={<About />} />
        </Route>
      </Routes>
    </BrowserRouter>
  </StrictMode>
)
```

## 6. 使用教学代码

### 6.1 复制示例代码

将 `教学代码` 文件夹中的示例组件复制到你的项目中：

```
教学代码/
├── 第3章-Hooks入门/
│   ├── 01-Counter.jsx
│   ├── 02-TodoList.jsx
│   ├── 03-UserList.jsx
│   ├── 04-CountdownTimer.jsx
│   └── 05-PomodoroTimer.jsx
└── ...
```

### 6.2 导入使用

```jsx
// 在 App.jsx 或其他页面中导入
import Counter from './components/01-Counter'
import TodoList from './components/02-TodoList'
import CountdownTimer from './components/04-CountdownTimer'
import PomodoroTimer from './components/05-PomodoroTimer'

function App() {
  return (
    <div>
      <h2>计数器示例</h2>
      <Counter />

      <h2>待办事项列表示例</h2>
      <TodoList />

      <h2>倒计时器示例</h2>
      <CountdownTimer />

      <h2>番茄钟示例</h2>
      <PomodoroTimer />
    </div>
  )
}
```

## 9. 常见问题

### 9.1 端口被占用

```bash
# 查看占用端口的进程
netstat -ano | findstr 5173

# 结束进程
taskkill /PID <进程ID> /F

# 或使用其他端口
pnpm dev -- --port 3001
```

### 9.2 依赖安装失败

```bash
# 清理缓存
rm -rf node_modules
rm pnpm-lock.yaml

# 重新安装
pnpm install
```

### 9.3 热更新不生效

```bash
# 清理 Vite 缓存
rm -rf node_modules/.vite

# 重新启动
pnpm dev
```
