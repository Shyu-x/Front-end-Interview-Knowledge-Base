# React 学习常用命令速查（2025最新版）

## 环境搭建

```bash
# ===== Node.js 25.x =====

# 验证安装
node -v    # v25.4.0
npm -v     # 10.5.0

# 使用 nvm 管理多版本 (Linux/Mac)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.0/install.sh | bash
nvm install 25
nvm use 25

# ===== pnpm =====

# 安装 pnpm
npm install -g pnpm
pnpm --version  # 9.x
```

## 项目创建

```bash
# ===== 使用 pnpm 创建项目（推荐）=====

# 创建 React 项目
pnpm create vite@latest my-app -- --template react
cd my-app
pnpm install

# 使用 TypeScript 模板
pnpm create vite@latest my-app -- --template react-ts

# ===== 使用 npm =====

npm create vite@latest my-app -- --template react
npm install

# ===== 使用 Yarn =====

yarn create vite my-app --template react
yarn
```

## 常用依赖安装

```bash
# 进入项目目录
cd my-app

# 状态管理
pnpm add zustand

# UI 组件库
pnpm add antd
pnpm add @ant-design/icons

# 路由
pnpm add react-router-dom

# HTTP 请求
pnpm add axios

# 日期处理
pnpm add dayjs

# 动画
pnpm add framer-motion

# 表单
pnpm add react-hook-form

# CSS 工具
pnpm add classnames
pnpm add sass

# 开发工具
pnpm add -D eslint prettier
```

## 运行命令

```bash
# 开发服务器
pnpm dev

# 指定端口
pnpm dev -- --port 3000

# 局域网访问
pnpm dev -- --host

# 生产构建
pnpm build

# 预览构建结果
pnpm preview

# 指定端口预览
pnpm preview -- --port 3000

# 运行测试
pnpm test

# 代码检查
pnpm lint
```

## Git 操作

```bash
# 初始化仓库
git init

# 添加文件
git add .
git add 文件名

# 提交
git commit -m "提交信息"

# 查看状态
git status

# 查看历史
git log
git log --oneline

# 创建分支
git checkout -b feature/new-feature

# 切换分支
git checkout main

# 合并分支
git merge feature/new-feature

# 删除分支
git branch -d feature/new-feature
```

## 常用 Snippets

### 函数组件 (rfc)
```jsx
import React from 'react'

function Component() {
  return (
    <div>
      Component
    </div>
  )
}

export default Component
```

### useState (us)
```jsx
const [state, setState] = useState()
```

### useEffect (ue)
```jsx
useEffect(() => {
  return () => {}
}, [])
```

### useRef (uref)
```jsx
const ref = useRef()
```

### useCallback
```jsx
const handleClick = useCallback(() => {
  // code
}, [deps])
```

### useMemo
```jsx
const value = useMemo(() => {
  return computeExpensiveValue(a, b)
}, [a, b])
```
